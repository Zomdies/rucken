# rucken

Console nx tools for rucken

## Tools

```
Usage: rucken tools [options] [command]

common console nx tools

Options:
  -h, --help              display help for command

Commands:
  files-list|fl       create list of ts files for all nx libraries
  version-updater|vu  update versions in all nx applications
  all|a               create list of ts files for all nx libraries + update versions in all nx applications
  help [command]          display help for command
```
